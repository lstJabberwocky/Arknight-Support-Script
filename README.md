# 明日方舟辅助脚本

## 项目概述

明日方舟界园肉鸽辅助项目,主要针对单调且机械式的动作,使用脚本进行辅助

未来计划加入roll钱概率计算脚本

## 依赖

- `pynput` — 鼠标控制（主要依赖，所有脚本均使用）
- `pyautogui` — 鼠标位置查询（大部分脚本使用）
- `keyboard` — F8 快捷键检测，用于紧急停止

## 架构


**坐标配置** (`coordinate.json`)：
- 存放命名的屏幕坐标  : choumou(筹谋),gusi(故肆),非固定的坐标,供多个脚本共享
- 部分脚本不使用此配置，未来进行重构将全面读取 coordinate.json 进行配置

**脚本文件**（每个脚本独立运行：`python <文件名>.py`）：
- 由于电脑端明日方舟窗口的原因, 脚本需要 `以管理员身份运行`   
- `良好的网路`可以显著提升脚本使用体验
- 所有脚本遵循相同的模式：5 秒启动延迟 → 带间隔的鼠标点击循环 → 紧急停止检查
- 硬编码的屏幕坐标与分辨率相关 , 若屏幕分辨率为 `2560 * 1600` , 可直接使用,否则需要配置坐标


## 紧急停止
**将鼠标移至 ` y >= 1538 `或者 按`F8` 将紧急退出**,部分脚本可能需要多试几次 , 若有必要可以切换到脚本运行窗口(vscode , pycharm, powershell...) , 按`ALT + F4` 强制关闭窗口,以达成退出目的
